# وب اپ پیش‌بینی قیمت مسکن

این پروژه یک وب اپ مبتنی بر ماشین لرنینگ و Streamlit است که با دریافت اطلاعات ملک، قیمت تقریبی آن را پیش‌بینی می‌کند.

## قابلیت‌ها

* پیش‌بینی قیمت مسکن با استفاده از مدل آموزش‌دیده
* رابط کاربری ساده و کاربرپسند
* پشتیبانی از ویژگی‌های مختلف ملک:

  * متراژ
  * تعداد اتاق
  * پارکینگ
  * انباری
  * آسانسور
  * موقعیت مکانی
* نمایش نتیجه به صورت لحظه‌ای

## تکنولوژی‌های استفاده شده

* Python
* Streamlit
* Scikit-Learn
* Pandas
* NumPy
* Joblib

## ساختار پروژه

├── app.py
├── model.pkl
├── requirements.txt
└── README.md

## نصب پروژه

ابتدا پروژه را دریافت کنید:

```bash
git clone https://github.com/mamadj0n/Tehran-House-Price-Predictor/new/main.git
cd Tehran-House-Price-Predictor
```

محیط مجازی بسازید:

```bash
python -m venv venv
```

فعال‌سازی محیط:

لینوکس:

```bash
source venv/bin/activate
```

ویندوز:

```bash
venv\Scripts\activate
```

نصب وابستگی‌ها:

```bash
pip install -r requirements.txt
```

## اجرای برنامه

```bash
streamlit run app.py
```

## ویژگی‌های ورودی مدل

| ویژگی     | توضیح             |
| --------- | ----------------- |
